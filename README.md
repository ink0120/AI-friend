### Installation
1. **Clone the repository**
   ```
   git clone https://github.com/ink0120/AI-friend.git
   ```
   

   Enter the folder

   
   
   ```
   cd ai-app
   ```
   

3. **Install dependancies**
  
   ```
    npm install
   ```

   
5. **Set your Gemini API key**

   
   Create a .env file in the root and paste this in (include your API key):

   
   ```
   VITE_GEMINI_API_KEY=your_actual_key_here
   ```

   
   
   (For how to create an API key, visit: [Gemini API creation](https://aistudio.google.com/apikey)


7. **Start the server**


    ```
   npm run dev
    ```
